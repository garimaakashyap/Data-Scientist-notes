## Introduction

### Key Differences from C++
- **No explicit type declarations** - Python is dynamically typed
- **Indentation matters** - Uses spaces/tabs instead of braces `{}`
- **No semicolons needed** - Newline indicates statement end
- **Everything is an object** - Even integers and functions
- **Automatic memory management** - No manual `new`/`delete`

### Basic Syntax
```python
# Single line comment

"""
Multi-line
comment (docstring)
"""

# Print statement
print("Hello, World!")

# Multiple statements on one line (not recommended)
x = 5; y = 10
```

## Data Types

### Numeric Types

```python
# Integer (unlimited size)
x = 42
y = -10
z = 0

# Complex numbers

str(123)       # "123"
bool(0)        # False

bool("")       # False
bool("abc")    # True
```

## Variables and Assignment

### Variable Naming
```python
# Valid names
variable_name = 10
_variable = 20
variable123 = 30
VariableName = 40  # Convention: PascalCase for classes

# Invalid names
# 123variable = 10  # Error: can't start with number
# var-name = 10     # Error: can't use hyphen
# var name = 10     # Error: can't use space
```

### Multiple Assignment
```python
# Single value to multiple variables
x = y = z = 0

# Multiple values to multiple variables
a, b, c = 1, 2, 3
x, y = y, x  # Swap values

# Unpacking
data = [1, 2, 3]
a, b, c = data

# Extended unpacking
first, *rest = [1, 2, 3, 4, 5]
# first = 1, rest = [2, 3, 4, 5]
```

## Operators

### Arithmetic Operators
```python
a = 10
b = 3

a + b    # 13 (addition)
a - b    # 7 (subtraction)
a * b    # 30 (multiplication)
a / b    # 3.333... (true division, always float)
a // b   # 3 (floor division)
a % b    # 1 (modulo)
a ** b   # 1000 (exponentiation)

# Augmented assignment
a += 1   # a = a + 1
a -= 1   # a = a - 1
a *= 2   # a = a * 2
a /= 2   # a = a / 2
```

### Comparison Operators
```python
a == b   # Equal
a != b   # Not equal
a < b    # Less than
a > b    # Greater than
a <= b   # Less than or equal
a >= b   # Greater than or equal
a is b   # Identity (same object)
a is not b  # Not identity
```

### Logical Operators
```python
x = True
y = False

x and y   # False (logical AND)
x or y    # True (logical OR)
not x     # False (logical NOT)

# Short-circuit evaluation
result = x and some_function()  # some_function() only called if x is True
```

### Membership Operators
```python
x in [1, 2, 3]      # True
x not in [1, 2, 3]  # False
"a" in "hello"      # False
"el" in "hello"     # True
```

### Bitwise Operators
```python
a = 5   # 101 in binary
b = 3   # 011 in binary

a & b   # 1 (AND) - 001
a | b   # 7 (OR) - 111
a ^ b   # 6 (XOR) - 110
~a      # -6 (NOT, two's complement)
a << 1  # 10 (left shift)
a >> 1  # 2 (right shift)
```

## Control Flow

### If-Else Statements
```python
# Simple if
if x > 0:
    print("Positive")

# If-else
if x > 0:
    print("Positive")
else:
    print("Non-positive")

# If-elif-else
if x > 0:
    print("Positive")
elif x < 0:
    print("Negative")
else:
    print("Zero")

# Ternary operator (conditional expression)
result = "positive" if x > 0 else "non-positive"
```

### For Loops
```python
# Iterate over range
for i in range(5):        # 0, 1, 2, 3, 4
    print(i)

for i in range(1, 6):     # 1, 2, 3, 4, 5
    print(i)

for i in range(0, 10, 2): # 0, 2, 4, 6, 8 (step by 2)
    print(i)

# Iterate over list
numbers = [1, 2, 3, 4, 5]
for num in numbers:
    print(num)

# Iterate with index
for i, num in enumerate(numbers):
    print(f"Index {i}: {num}")

# Iterate over dictionary
d = {"a": 1, "b": 2, "c": 3}
for key in d:
    print(key, d[key])

for key, value in d.items():
    print(key, value)
```

### While Loops
```python
# Basic while loop
count = 0
while count < 5:
    print(count)
    count += 1

# While-else (else executes if loop completes normally)
count = 0
while count < 5:
    print(count)
    count += 1
else:
    print("Loop completed")

# Break and continue
while True:
    value = input("Enter a number: ")
    if value == "quit":
        break  # Exit loop
    if value == "skip":
        continue  # Skip to next iteration
    print(value)
```

### Loop Control
```python
# Break - exit loop immediately
for i in range(10):
    if i == 5:
        break
    print(i)  # Prints 0-4

# Continue - skip to next iteration
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)  # Prints odd numbers only

# Pass - do nothing (placeholder)
for i in range(10):
    if i == 5:
        pass  # Do nothing
    else:
        print(i)
```

## Strings

### String Creation
```python
# Single quotes
s1 = 'Hello'

# Double quotes
s2 = "World"

# Triple quotes (multi-line)
s3 = """This is
a multi-line
string"""

# Raw strings (no escape processing)
path = r"C:\Users\Name\Documents"

# f-strings (formatted string literals) - Python 3.6+
name = "Alice"
age = 30
message = f"My name is {name} and I'm {age} years old"

# Format method
message = "My name is {} and I'm {} years old".format(name, age)
message = "My name is {name} and I'm {age} years old".format(name=name, age=age)
```

### String Operations
```python
s = "Hello, World!"

# Length
len(s)  # 13

# Indexing
s[0]    # 'H' (first character)
s[-1]   # '!' (last character)

# Slicing
s[0:5]      # "Hello" (from index 0 to 4)
s[:5]       # "Hello" (from start to 4)
s[7:]       # "World!" (from 7 to end)
s[::2]      # "Hlo ol!" (every 2nd character)
s[::-1]     # "!dlroW ,olleH" (reverse)

# Concatenation
"Hello" + " " + "World"  # "Hello World"

# Repetition
"Hi" * 3  # "HiHiHi"

# Membership
"World" in s  # True
"Python" in s  # False
```

### String Methods
```python
s = "  Hello, World!  "

s.lower()        # "  hello, world!  "
s.upper()        # "  HELLO, WORLD!  "
s.strip()        # "Hello, World!" (remove leading/trailing whitespace)
s.lstrip()       # Remove leading whitespace
s.rstrip()       # Remove trailing whitespace
s.replace("World", "Python")  # "  Hello, Python!  "
s.split(",")     # ['  Hello', ' World!  ']
s.find("World")  # 8 (index of first occurrence, -1 if not found)
s.count("l")     # 3 (count occurrences)
s.startswith("Hello")  # False (has leading spaces)
s.endswith("!")  # True
s.isalpha()      # False (contains non-letters)
s.isdigit()      # False
s.isalnum()      # False
```

## Lists

### List Creation
```python
# Empty list
empty = []

# List with elements
numbers = [1, 2, 3, 4, 5]
mixed = [1, "hello", 3.14, True]
nested = [[1, 2], [3, 4], [5, 6]]

# List constructor
numbers = list(range(5))  # [0, 1, 2, 3, 4]
```

### List Operations
```python
lst = [1, 2, 3]

# Indexing
lst[0]      # 1
lst[-1]     # 3 (last element)

# Slicing
lst[1:3]    # [2, 3]
lst[:2]     # [1, 2]
lst[1:]     # [2, 3]
lst[::-1]   # [3, 2, 1] (reverse)

# Length
len(lst)    # 3

# Concatenation
lst + [4, 5]  # [1, 2, 3, 4, 5]

# Repetition
[0] * 5     # [0, 0, 0, 0, 0]

# Membership
2 in lst    # True
5 in lst    # False
```

### List Methods
```python
lst = [1, 2, 3]

# Adding elements
lst.append(4)           # [1, 2, 3, 4]
lst.insert(1, 10)       # [1, 10, 2, 3, 4] (insert at index 1)
lst.extend([5, 6])      # [1, 10, 2, 3, 4, 5, 6]

# Removing elements
lst.remove(10)          # Remove first occurrence of 10
lst.pop()               # Remove and return last element (6)
lst.pop(1)              # Remove and return element at index 1
del lst[0]              # Delete element at index 0
lst.clear()             # Remove all elements

# Finding elements
lst = [1, 2, 3, 2]
lst.index(2)            # 1 (index of first occurrence)
lst.count(2)            # 2 (count occurrences)

# Sorting
lst = [3, 1, 4, 1, 5]
lst.sort()              # [1, 1, 3, 4, 5] (in-place)
sorted_lst = sorted(lst)  # Returns new sorted list
lst.sort(reverse=True)  # [5, 4, 3, 1, 1]

# Reversing
lst.reverse()           # Reverse in-place
reversed_lst = lst[::-1]  # Returns new reversed list
```

## Tuples

### Tuple Creation
```python
# Empty tuple
empty = ()

# Tuple with elements (parentheses optional)
t1 = (1, 2, 3)
t2 = 1, 2, 3  # Same as above
single = (1,)  # Single element tuple (comma required)

# Tuple constructor
t = tuple([1, 2, 3])  # (1, 2, 3)
```

### Tuple Operations
```python
t = (1, 2, 3)

# Indexing and slicing (same as lists)
t[0]      # 1
t[1:3]    # (2, 3)

# Immutable - cannot modify
# t[0] = 10  # Error!

# Packing and unpacking
point = (3, 4)
x, y = point  # Unpacking

# Returning multiple values
def get_name_age():
    return "Alice", 30

name, age = get_name_age()
```

### Why Tuples?
- **Immutable** - Can't be changed after creation
- **Faster** - More efficient than lists
- **Hashable** - Can be used as dictionary keys
- **Return multiple values** - Common pattern in Python

## Dictionaries

### Dictionary Creation
```python
# Empty dictionary
empty = {}

# Dictionary with key-value pairs
d = {"name": "Alice", "age": 30, "city": "New York"}
d = dict(name="Alice", age=30, city="New York")  # Same

# Dictionary from lists
keys = ["a", "b", "c"]
values = [1, 2, 3]
d = dict(zip(keys, values))  # {'a': 1, 'b': 2, 'c': 3}
```

### Dictionary Operations
```python
d = {"name": "Alice", "age": 30}

# Accessing values
d["name"]           # "Alice"
d.get("name")       # "Alice"
d.get("city", "Unknown")  # "Unknown" (default if key doesn't exist)

# Adding/Updating
d["city"] = "New York"  # Add new key-value
d["age"] = 31           # Update existing value
d.update({"city": "Boston", "country": "USA"})  # Update multiple

# Removing
del d["city"]       # Delete key-value pair
value = d.pop("age")  # Remove and return value
d.clear()           # Remove all items

# Keys, values, items
d.keys()            # dict_keys(['name', 'age'])
d.values()          # dict_values(['Alice', 30])
d.items()           # dict_items([('name', 'Alice'), ('age', 30)])

# Membership
"name" in d         # True
"Alice" in d.values()  # True

# Length
len(d)              # 2
```

### Dictionary Comprehension
```python
# Create dictionary from iterable
squares = {x: x**2 for x in range(5)}  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

# With condition
even_squares = {x: x**2 for x in range(10) if x % 2 == 0}
```

## Sets

### Set Creation
```python
# Empty set (note: {} is empty dict, not set)
empty = set()

# Set with elements
s = {1, 2, 3, 4, 5}
s = set([1, 2, 3, 4, 5])  # From list

# Set removes duplicates
s = {1, 2, 2, 3, 3, 3}  # {1, 2, 3}
```

### Set Operations
```python
s1 = {1, 2, 3, 4}
s2 = {3, 4, 5, 6}

# Union
s1 | s2           # {1, 2, 3, 4, 5, 6}
s1.union(s2)      # Same

# Intersection
s1 & s2           # {3, 4}
s1.intersection(s2)  # Same

# Difference
s1 - s2           # {1, 2}
s1.difference(s2)  # Same

# Symmetric difference
s1 ^ s2           # {1, 2, 5, 6}
s1.symmetric_difference(s2)  # Same

# Membership
1 in s1           # True

# Adding/Removing
s1.add(5)         # Add element
s1.remove(1)      # Remove element (raises error if not found)
s1.discard(1)     # Remove element (no error if not found)
s1.pop()          # Remove and return arbitrary element
s1.clear()        # Remove all elements
```

### Set Methods
```python
s1 = {1, 2, 3}
s2 = {2, 3, 4}

s1.issubset(s2)        # False
s1.issuperset(s2)      # False
s1.isdisjoint(s2)      # False (has common elements)
```

## List Comprehensions

### Basic Syntax
```python
# Traditional way
squares = []
for x in range(10):
    squares.append(x**2)

# List comprehension
squares = [x**2 for x in range(10)]

# With condition
even_squares = [x**2 for x in range(10) if x % 2 == 0]

# Nested loops
matrix = [[i*j for j in range(3)] for i in range(3)]
# [[0, 0, 0], [0, 1, 2], [0, 2, 4]]
```

### Dictionary and Set Comprehensions
```python
# Dictionary comprehension
squares_dict = {x: x**2 for x in range(5)}

# Set comprehension
squares_set = {x**2 for x in range(5)}
```

## Input/Output

### Input
```python
# Reading input (always returns string)
name = input("Enter your name: ")
age = int(input("Enter your age: "))  # Convert to int

# Reading multiple values
x, y = map(int, input().split())  # "1 2" -> x=1, y=2
```

### Output
```python
# Print function
print("Hello")
print("World")  # Prints on new line

# Print without newline
print("Hello", end=" ")
print("World")  # "Hello World"

# Print multiple values
print("Name:", name, "Age:", age)
print(f"Name: {name}, Age: {age}")  # f-string (recommended)

# Formatting
print("Value: {:.2f}".format(3.14159))  # "Value: 3.14"
print(f"Value: {3.14159:.2f}")  # Same with f-string
```

### File I/O
```python
# Writing to file
with open("file.txt", "w") as f:
    f.write("Hello, World!\n")
    f.write("Second line\n")

# Reading from file
with open("file.txt", "r") as f:
    content = f.read()        # Read entire file
    lines = f.readlines()     # Read all lines as list
    line = f.readline()       # Read one line

# Reading line by line
with open("file.txt", "r") as f:
    for line in f:
        print(line.strip())  # strip() removes newline

# Appending to file
with open("file.txt", "a") as f:
    f.write("New line\n")
```

## Common Patterns

### Swapping Variables
```python
a, b = b, a  # Pythonic way
```

### Multiple Return Values
```python
def get_stats(numbers):
    return min(numbers), max(numbers), sum(numbers) / len(numbers)

minimum, maximum, average = get_stats([1, 2, 3, 4, 5])
```

### Checking Empty Collections
```python
# Good way
if not my_list:
    print("List is empty")

# Also works
if len(my_list) == 0:
    print("List is empty")
```

### Iterating with Index
```python
# Using enumerate
for i, value in enumerate(my_list):
    print(f"Index {i}: {value}")
```

### Iterating Over Dictionary
```python
# By keys (default)
for key in my_dict:
    print(key, my_dict[key])

# By items (recommended)
for key, value in my_dict.items():
    print(key, value)
```

## Quick Reference: C++ vs Python

| C++ | Python |
|-----|--------|
| `int x = 5;` | `x = 5` |
| `if (x > 0) { ... }` | `if x > 0: ...` |
| `for (int i = 0; i < 10; i++)` | `for i in range(10):` |
| `vector<int> v;` | `v = []` |
| `v.push_back(x);` | `v.append(x)` |
| `map<string, int> m;` | `m = {}` |
| `m["key"] = value;` | `m["key"] = value` |
| `#include <iostream>` | `import module_name` |
| `cout << x << endl;` | `print(x)` |
| `cin >> x;` | `x = input()` |
