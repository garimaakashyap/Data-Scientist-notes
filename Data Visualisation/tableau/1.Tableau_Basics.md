# Tableau Basics - Complete Guide

## Introduction
Tableau is a powerful data visualization tool that helps you create interactive dashboards and reports. It's widely used in business intelligence and data analysis.

## What is Tableau?
Tableau is a visual analytics platform that transforms data into actionable insights through interactive dashboards and visualizations.

## Tableau Products

1. **Tableau Desktop** - Create visualizations and dashboards
2. **Tableau Server** - Share and collaborate on Tableau content
3. **Tableau Online** - Cloud-based sharing and collaboration
4. **Tableau Public** - Free version with public sharing
5. **Tableau Prep** - Data preparation tool

## Installation

### Download Tableau Desktop
1. Go to https://www.tableau.com/products/desktop
2. Download Tableau Desktop
3. Install and launch the application
4. Activate with license or use trial

## 1. Getting Started

**Main Components:**
- **Data Pane** - Lists all data sources and fields
- **Shelves** - Rows, Columns, Marks, Filters, Pages
- **Canvas** - Where visualizations are created
- **Show Me** - Suggests chart types

**Step 1: Connect to Data Source**
1. Click "Connect to Data"

**Step 2: Select Data**

**Step 3: Start Analysis**
- Drag fields to canvas
- Create visualizations

## 2. Understanding Data Types

### Field Types

**Dimensions (Blue):**
- Categorical data
- Text, dates, geographic data
- Used for grouping and filtering
- Examples: Product Name, Date, Region

**Measures (Green):**
- Numerical data
- Can be aggregated (sum, average, etc.)
- Examples: Sales, Quantity, Profit

### Data Roles

**Discrete (Blue):**
- Individual values
- Creates headers
- Examples: Product names, Categories

**Continuous (Green):**
- Range of values
- Creates axes
- Examples: Sales amounts, Dates

### Changing Field Types

**To Change:**
1. Right-click field in Data pane
2. Select "Convert to Dimension" or "Convert to Measure"
3. Or drag field to different area

## 3. Basic Visualizations

### Bar Chart

**How to Create:**
1. Drag dimension to Columns
2. Drag measure to Rows
3. Tableau creates bar chart automatically

**Example:**
- Columns: Category
- Rows: Sales
- Result: Bar chart showing sales by category

### Line Chart

**How to Create:**
1. Drag date to Columns
2. Drag measure to Rows
3. Tableau creates line chart

**Example:**
- Columns: Order Date (continuous)
- Rows: Sales
- Result: Line chart showing sales over time

### Scatter Plot

**How to Create:**
1. Drag measure to Columns
2. Drag measure to Rows
3. Drag dimension to Color (optional)
4. Tableau creates scatter plot

**Example:**
- Columns: Sales
- Rows: Profit
- Color: Region
- Result: Scatter plot showing relationship

### Pie Chart

**How to Create:**
1. Drag dimension to Columns
2. Drag measure to Rows
3. Click "Show Me" → Select Pie Chart
4. Drag dimension to Color

**Example:**
- Dimension: Category
- Measure: Sales
- Result: Pie chart showing sales distribution

### Table

**How to Create:**
1. Drag dimensions to Rows
2. Drag measures to Columns
3. Creates crosstab table

**Example:**
- Rows: Product, Region
- Columns: Sales, Profit
- Result: Table with sales and profit by product and region

## 6. Filters

**1. Quick Filters:**
- Right-click field → Show Filter

- Faster performance

**Top Tab:**
- Top N
- Bottom N

- By formula

## 8. Groups and Sets

### Groups

**What are Groups?**
- Combine related members
- Simplify visualization
- Create categories

**How to Create:**
1. Select members in visualization
2. Right-click → Group
3. Name the group
4. Use in analysis

**Example:**
- Group: "High Sales" (Products with sales > $1000)
- Group: "Low Sales" (Products with sales < $500)

### Sets

**What are Sets?**
- Custom fields based on conditions
- Dynamic or static
- More flexible than groups

**How to Create:**
1. Right-click field → Create → Set
2. Choose members or condition
3. Name the set
4. Use in visualization

**Types of Sets:**
- **Fixed Set:** Specific members
- **Dynamic Set:** Based on condition
- **Computed Set:** Based on formula

## 9. Calculated Fields

### What are Calculated Fields?
Custom fields created using formulas and existing fields.

### Creating Calculated Fields

**Steps:**
1. Right-click Data pane → Create Calculated Field
2. Enter formula
3. Name the field
4. Click OK

### Basic Formulas

**Mathematical Operations:**
```
Profit = [Sales] - [Cost]
Total = [Quantity] * [Price]
Percentage = [Sales] / [Total Sales]
```

**String Functions:**
```
Full Name = [First Name] + " " + [Last Name]
Upper Case = UPPER([Product Name])
```

**Date Functions:**
```
Year = YEAR([Order Date])
Month Name = MONTHNAME([Order Date])
Days Since = TODAY() - [Order Date]
```

**Logical Functions:**
```
High Sales = IF [Sales] > 1000 THEN "High" ELSE "Low" END
Category = CASE [Region]
    WHEN "North" THEN "North Region"
    WHEN "South" THEN "South Region"
    ELSE "Other"
END
```

### Common Functions

**Aggregation:**
- SUM()
- AVG()
- COUNT()
- MIN()
- MAX()

**String:**
- UPPER()
- LOWER()
- TRIM()
- LEFT()
- RIGHT()
- MID()

**Date:**
- YEAR()
- MONTH()
- DAY()
- DATE()
- DATEDIFF()

**Logical:**
- IF THEN ELSE END
- CASE WHEN THEN END
- ISNULL()
