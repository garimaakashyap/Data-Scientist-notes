# Pandas Basics - Simple Guide

## What is Pandas?
Pandas is a library for working with data tables (like Excel spreadsheets). It makes data analysis easy and powerful.

## Installation
```python
pip install pandas
```

## Import Pandas
```python
import pandas as pd
```

## 1. Creating DataFrames

### From a Dictionary
```python
# Create a DataFrame (like a table)
data = {
    'Name': ['Alice', 'Bob', 'Charlie', 'Diana'],
    'Age': [25, 30, 35, 28],
    'City': ['New York', 'London', 'Tokyo', 'Paris']
}

df = pd.DataFrame(data)
print(df)
#      Name  Age      City
# 0   Alice   25  New York
# 1     Bob   30    London
# 2  Charlie   35     Tokyo
# 3   Diana   28     Paris
```

### From a List of Lists
```python
data = [['Alice', 25], ['Bob', 30], ['Charlie', 35]]
df = pd.DataFrame(data, columns=['Name', 'Age'])
print(df)
```

### From CSV File
```python
# Read CSV file
df = pd.read_csv('data.csv')

# Display first few rows
print(df.head())
```

## 2. Creating Series

A Series is like a single column (1D array with labels).

```python
# Create a Series
ages = pd.Series([25, 30, 35, 28], name='Age')
print(ages)
# 0    25
# 1    30
# 2    35
# 3    28
# Name: Age, dtype: int64
```

## 3. Viewing Data

```python
df = pd.DataFrame({
    'Name': ['Alice', 'Bob', 'Charlie', 'Diana', 'Eve'],
    'Age': [25, 30, 35, 28, 32],
    'Salary': [50000, 60000, 70000, 55000, 65000]
})

# First 5 rows
print(df.head())

# Last 5 rows
print(df.tail())

# First 3 rows
print(df.head(3))

# Basic information
print(df.info())        # Data types and memory usage
print(df.describe())    # Statistical summary
print(df.shape)         # (rows, columns)
print(df.columns)       # Column names
```

## 4. Selecting Data

### Select Columns
```python
# Select one column (returns Series)
ages = df['Age']
print(ages)

# Select multiple columns (returns DataFrame)
subset = df[['Name', 'Age']]
print(subset)
```

### Select Rows
```python
# Select by index
first_row = df.iloc[0]        # First row
first_three = df.iloc[0:3]    # First 3 rows

# Select by label (if index has names)
df_indexed = df.set_index('Name')
alice_data = df_indexed.loc['Alice']
```

### Select Specific Cell
```python
# By position
value = df.iloc[0, 1]  # Row 0, Column 1

# By label
value = df.loc[0, 'Age']  # Row 0, Column 'Age'
```

## 5. Filtering Data

### Simple Conditions
```python
# People older than 30
older = df[df['Age'] > 30]
print(older)

# People in specific city
ny_people = df[df['City'] == 'New York']
print(ny_people)

# Multiple conditions (use & for AND, | for OR)
filtered = df[(df['Age'] > 30) & (df['Salary'] > 60000)]
print(filtered)
```

### Using isin()
```python
# People in specific cities
cities = ['New York', 'London']
filtered = df[df['City'].isin(cities)]
print(filtered)
```

## 6. Adding and Removing Columns

### Add Column
```python
# Add new column
df['Bonus'] = df['Salary'] * 0.1
print(df)

# Add column based on condition
df['Status'] = df['Age'].apply(lambda x: 'Senior' if x > 30 else 'Junior')
print(df)
```

### Remove Column
```python
# Drop column
df = df.drop('Bonus', axis=1)  # axis=1 means column

# Drop multiple columns
df = df.drop(['Bonus', 'Status'], axis=1)
```

## 7. Basic Operations

### Mathematical Operations
```python
# Add columns
df['Total'] = df['Salary'] + df['Bonus']

# Multiply
df['Double_Salary'] = df['Salary'] * 2

# Statistical operations
print(df['Salary'].mean())    # Average
print(df['Salary'].sum())     # Sum
print(df['Salary'].max())     # Maximum
print(df['Salary'].min())     # Minimum
print(df['Salary'].std())     # Standard deviation
```

### String Operations
```python
# Convert to uppercase
df['Name_Upper'] = df['Name'].str.upper()

# Find length
df['Name_Length'] = df['Name'].str.len()

# Check if contains
has_a = df['Name'].str.contains('a', case=False)
print(has_a)
```

## 8. Handling Missing Data

```python
# Check for missing values
print(df.isnull())        # True where missing
print(df.isnull().sum())  # Count missing per column

# Drop rows with missing values
df_clean = df.dropna()

# Fill missing values
df_filled = df.fillna(0)           # Fill with 0
df_filled = df.fillna(df.mean())   # Fill with mean
```

## 9. Sorting Data

```python
# Sort by one column
sorted_df = df.sort_values('Age')
print(sorted_df)

# Sort by multiple columns
sorted_df = df.sort_values(['Age', 'Salary'], ascending=[True, False])

# Sort by index
sorted_df = df.sort_index()
```

## 10. Basic Grouping

```python
# Group by column and calculate mean
grouped = df.groupby('City')['Salary'].mean()
print(grouped)

# Multiple aggregations
grouped = df.groupby('City').agg({
    'Salary': ['mean', 'sum', 'count'],
    'Age': 'mean'
})
print(grouped)
```

## 11. Saving Data

```python
# Save to CSV
df.to_csv('output.csv', index=False)

# Save to Excel
df.to_excel('output.xlsx', index=False)

# Save to JSON
df.to_json('output.json')
```

## Key Takeaways
- DataFrame = table with rows and columns
- Series = single column
- Use `df['column']` to select columns
- Use `df[condition]` to filter rows
- Pandas makes data analysis much easier than regular Python
